Dynamic tabs

Here is the process for clarification:
Add data-toggle="tab" attribute to each link. Then add a .tab-pane class with a unique ID for every tab and wrap them inside a div element with class .tab-content.
Add .fade class to .tab-pane for fade-in-out.
